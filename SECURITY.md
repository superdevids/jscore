# Security Policy

## Reporting a Vulnerability

Report vulnerabilities via email: adityasuperdev@gmail.com

Include: description, reproduction steps, affected versions, impact.
We will respond within 48 hours.

## Supported Versions

| Version | Support |
|---------|---------|
| 0.8.x | Active |
| 0.7.x | Limited |
| < 0.7 | Not supported |
